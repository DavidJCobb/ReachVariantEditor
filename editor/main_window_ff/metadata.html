Metadata

The metadata tab allows you to edit a file's title, description, timestamps, and authorship data.

The fields available to edit are as follows:

Name
The filename displayed in-game for this game variant.
Description
The description displayed in-game for this game variant.
Author

Game variants embed the Xbox LIVE gamertag and Xbox User ID (XUID) of the player who created them. However, Halo: The Master Chief Collection does not make use of the embedded gamertag if you are connected to Xbox LIVE. A file that has an embedded gamertag but no XUID will be displayed as having been created by 343 Industries.

Embedded gamertags are limited to 15 ASCII characters. The special character ¦ is used to denote a game variant created by Bungie or 343 Industries.

A button is provided to let you strip out the author XUID embedded in the file. However, you are not able to associate a file with arbitrary XUIDs. If you wish to add your own gamertag as an author, you'll have to resave the file in-game.

Created At
The date and time at which the file was created.
Editor
Game variants embed the Xbox LIVE gamertag and Xbox User ID (XUID) of the player who edited them (if anyone did) after they were created. This data works the same as authorship data.
Edited At
The date and time at which the file was edited.

Notes

Official game variants may have descriptions that look like $hr_gvar_Assault_desc. These descriptions are special codes that Halo: The Master Chief Collection can detect and replace with its own text. 343 Industries uses this functionality to retain a greater degree of control over how gametype descriptions are translated into different languages.
