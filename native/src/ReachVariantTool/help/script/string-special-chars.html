Special string characters

Halo: Reach's fonts include several unique icons and symbols, which are listed below. Note that the game uses two sets of fonts. "TV Nord" is used for large portions of the UI, including the scoreboard, HUD action prompts kill feed, and round cards. "Eurostile" is used for most of the rest of the HUD, including the game variant name, all scripted HUD widgets, and location names near the motion tracker.
