team.has_alliance_status

This function checks a team's alliance status toward another team.

Arguments

other
No description available.
status

Any of the following words:

  • friendly
  • neutral
  • enemy

Notes

  • During the Halo: Reach beta, Invasion implemented fireteams as actual entire teams: instead of having two teams invisibly divided into three pairs of two players, the game had six teams, and the teams' alliance statuses were altered to ally them with each other.
