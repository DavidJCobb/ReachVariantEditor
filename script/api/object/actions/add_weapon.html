object.add_weapon

Give a weapon to a biped, even if it isn't player-controlled.

Arguments

weapon

A type of object that can exist in the game world.

mode

One of the following values:

primary
TBA
secondary
TBA
force
Forcibly adds the weapon to the player and switches them to use it.

Example

global.object[0].add_weapon(assault_rifle, force)

Notes

  • If the biped is player-controlled and the player's Weapon Pickup trait is set to Disabled, then you must use the "force" mode to add the weapon.

  • Remember that this function works differently from player.add_weapon: the object function takes an object type and spawns a new weapon; while the player function takes an existing weapon object.
